Differential Loading in Angular Luis Aviles

Differential Loading hace referencia al proceso mediante el cual el navegador elige cargar entre JavaScript moderno o antiguo en función de sus propias capacidades. Con la nueva versión de Angular se puede aprovechar esta característica de forma predeterminada, al completar una compilación para JavaScript ES6/ES5 de tu aplicación. Al momento de cargar tu app, los usuarios obtienen automáticamente la versión de JavaScript que necesitan, permitiendo que el proceso de carga sea aún más rápido en navegadores modernos.
